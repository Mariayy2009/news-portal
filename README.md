# Новостной портал

## Тема проекта
Новостной портал (без backend)

## Описание
Простой адаптивный новостной портал, созданный с использованием семантического HTML и модульного CSS. Сайт содержит 4 страницы с единым визуальным стилем и навигацией.

## Страницы сайта
1. Главная страница (`index.html`) — список последних новостей
2. Категории (`pages/category.html`) — новости по категориям
3. Статья (`pages/article.html`) — полный текст новости
4. Контакты (`pages/contacts.html`) — контактная информация

## Используемые технологии
- HTML5 (семантическая верстка)
- CSS3 (Flexbox, Grid, адаптивный дизайн)
- Архитектура CSS (reset, base, layout, components)

## Запуск проекта
1. Склонируйте репозиторий
2. Откройте файл `index.html` в браузере
