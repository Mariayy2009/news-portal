# Практическая работа
Дисциплина: Проектирование и обеспечение бесперебойной работы web-сайта
Тема: Разработка многостраничного web-сайта на HTML и CSS с использованием архитектуры CSS и GitHub
Цель работы
Закрепить практические навыки разработки многостраничных web-сайтов с использованием семантического HTML и адаптивного CSS, сформировать понимание архитектуры
CSS, структуры проекта, навигации, единого визуального стиля, а также освоить базовые
навыки работы с системой контроля версий Git и платформой GitHub.

## Тема проекта
Новостной портал (без backend)

## Описание
Простой адаптивный новостной портал, созданный с использованием семантического HTML и модульного CSS. Сайт содержит 4 страницы с единым визуальным стилем и навигацией.

## Страницы сайта
1. Главная страница (`index.html`) — список последних новостей
2. Категории (`pages/category.html`) — новости по категориям
3. Статья (`pages/article.html`) — полный текст новости
4. Контакты (`pages/contacts.html`) — контактная информация

## Используемые технологии
- HTML5 (семантическая верстка)
- CSS3 (Flexbox, Grid, адаптивный дизайн)
- Архитектура CSS (reset, base, layout, components)
